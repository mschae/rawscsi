# Rawscsi

Rails Amazon Web Services Cloud Search Interface

## Installation

Add this line to your application's Gemfile:

    gem 'rawscsi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rawscsi

## Usage

Instantiate the service object:

my_search_helper = Rawsci::SearchHelper.new(:)
